* Novas Versões e Contribuições
- O desenvolvimento do projeto é feito em iterações incrementais (vX.X). O código de cada versão está disponível no diretório [[file:src/vX.X][src/vX.X]].
- Se este projeto tem como número de versão 1.0 a nova versão deverá ser 2.0 e assim sucessivamente.
- Na descrição do projeto informe que trata-se de uma nova versão, que tem como base a versão anterior e **DESTAQUE COM CLAREZA E OBJETIVIDADE QUAIS AS NOVIDADES DA NOVA VERSÃO**.
- Os nomes dos autores originais deve ser preservada na nova versão.
- Entre em contato com o Prof. André Duarte Bueno <bueno@lenep.uenf.br> para que o mesmo gere um novo projeto no GitHub do LDSC para que você (e sua equipe) possam trabalhar na nova versão.
